# Nyx 1.0.0 — initial release

The first Nyx: a glass new-tab replacement.

## Added
- **Link groups / boards** with tabs (home / projects / learning / personal).
- **Command mode** — type `>` in the search bar for commands.
- **Alt+D dashboard** — weather, profile, calendar, system stats, and a local
  music player.
- **Wallpapers** — image & video, with automatic accent-color extraction.
- **Most-visited row** from top sites.
- **Inline editing** — add boards/links via **+**, hover ⋯ menus, right-click
  context menus, and drag-to-reorder.
- **Stash** (saved things), a **password vault**, and a **phone companion**.
- Optional **`NullTab.MusicServer`** C# backend — music streaming, system stats,
  vault, and Discord Rich Presence.
