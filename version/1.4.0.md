# Nyx 1.4.0 — 2026-07-20

The biggest update yet: the extension gains **real bookmarks**, **translation**,
**history-aware search**, **display settings**, and a completely reworked
**password vault** that now reads as a security dashboard. Plus a wave of desktop
companion features.

> ⚠️ **After updating, reload the extension** in `brave://extensions`. This
> release requests new permissions — **`bookmarks`**, **`history`**, and the
> **`translate.googleapis.com`** host — and Chrome/Brave will prompt for them.

---

## Extension

### Added
- **Translation everywhere**
  - Select text on any page → the selection pill's new **Translate** button shows
    the translation inline, with the detected source language and a
    target-language picker that's remembered.
  - **`>translate` / `>tr`** command in the new-tab palette (result copied to the
    clipboard).
  - **Translate panel** (Alt+G): a full translator — type/paste on the left, live
    translation on the right, source + target pickers, and a **swap** button.
  - Uses Google's free endpoint through the background worker (no API key, no CORS
    issues) and auto-detects the source language.
- **History-aware search bar** — typing now suggests from your **browsing
  history** and **bookmarks** alongside your saved links, like the real address
  bar: ranked, deduped by URL, with favicons and a `saved` / `history` tag.
- **Settings → Display** — **Size** (S / M / L, scales the boards), **Weight**
  (Normal / Bold), **Board width** slider, and **Open links in a new tab**. All
  persist and apply on every new tab.
- **"What's new" popup** — after an update, the first new tab shows a one-time
  card with this version's highlights (dismissed with "Got it", remembered per
  version). Update its `items` in `js/whatsnew.js` each release.

### Changed
- **Saved tab → real bookmarks browser** — reads your actual browser bookmarks
  (`chrome.bookmarks`) instead of a private store that could be wiped, so nothing
  is ever lost. Shows **folders** (with a recursive count), a clickable
  **breadcrumb** + up button, and search across every bookmark. Adding a link
  bookmarks it into the current folder; an empty link creates a folder.
- **Vault → security dashboard** — a two-column layout (620 → 1080px) with an
  insight sidebar: a big **security-score ring**, a **breakdown** (all / strong /
  weak / reused / old) where each row **filters the list**, a **2FA coverage**
  bar, **recently used**, and **recently saved**, plus a **search** box. Every
  login shows a **favicon** and a colored **risk strip** (red = weak, amber =
  reused); crisp SVG icons; accent-glow header and tabs.
  - "Recently used" is powered by a new per-entry **last-used** timestamp,
    recorded on reveal and stored **inside the encrypted vault**.
- **Text-selection popup → small pill** — selecting text shows a compact
  "🔖 Save ▾" pill instead of a large card; **Save** stashes in one click, the
  **▾** opens a small category menu.

### Fixed
- **Vault dashboard layout** — the password list no longer leaves empty space
  below it, and the insight sidebar no longer gets its bottom cut off: the
  dashboard is now a fixed-height two-pane where each column scrolls on its own.

## Desktop companion (Nyx Vinyl HUD + backend + licensing)

### Added
- **Floating HUD touches** — album-art **ambient glow** (bass-reactive), **synced
  lyrics** (LRCLIB), and a stylized **waveform seek bar**.
- **Player SETTINGS tab + full-screen "Now Playing"** — live, persisted toggles
  for the glow, lyrics, waveform, visualizer, and always-on-top, plus an immersive
  Caelestia-style fullscreen dashboard bound to the same playback.
- **Control panel — four pages**: Library & paths, HUD, Diagnostics, About &
  updates (with a runtime-configurable activation server via `nyx.config.json`).
- **Free-plan usage cap** — an unlicensed install runs the bundled backend
  **2 hours/day**; a license lifts the cap. Persisted, resets at midnight.
- **License server** — Gumroad webhook mints + emails keys; per-product filter;
  SMTP; full run/deploy/setup docs.

### Changed
- **HUD volume controls the player only**, not the Windows master volume, and is
  persisted.
