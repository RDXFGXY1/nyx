# Nyx 1.4.0 — Release Notes

This release makes Nyx feel more powerful and polished than ever. The biggest
improvements focus on search, translation, bookmarks, password security, and the
desktop companion experience.

> ⚠️ After updating, please reload the extension in `brave://extensions`.
> This release requests new permissions for bookmarks, browsing history, and
> Google Translate access, so your browser may prompt you to allow them.

---

## What’s New

### New in the extension
- Translation everywhere: select text on any page and translate it inline, use
  the new `>translate` / `>tr` command, or open the full translator panel with
  Alt+G.
- History-aware search: the new-tab search now suggests results from your
  browsing history and bookmarks alongside saved links.
- Display settings: choose link size, weight, board width, and whether links
  should open in a new tab.
- “What’s new” popup: after updating, Nyx shows a one-time release summary so
  you can quickly see what changed.
- Update notifier: installed copies can now alert you when a newer version is
  available and guide you to the release.

### New in the desktop companion
- Floating Vinyl HUD upgrades: ambient album-art glow, synced lyrics, and a
  waveform seek bar.
- A new Player Settings tab with toggles for glow, lyrics, waveform, visualizer,
  and always-on-top, plus a full-screen “Now Playing” experience.
- New control panel pages for Library & Paths, HUD, Diagnostics, and About &
  Updates.
- Free-plan daily usage limits for the desktop app, with a clear warning and
  automatic stop once the limit is reached.
- Licensing and deployment improvements for the desktop server.

---

## Improvements and Fixes

- The Saved tab is now a real bookmarks browser that reads your actual browser
  bookmarks instead of relying on a private store.
- The Vault has been redesigned into a security dashboard with a clearer layout,
  risk scoring, filtering, search, and better visual feedback.
- The text-selection popup was reduced to a smaller, less intrusive pill.
- HUD volume now controls the player only and is persisted correctly.
- Several layout and UI issues were cleaned up to make the experience feel more
  stable and polished.
