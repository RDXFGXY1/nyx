# Nyx 1.5.0 — Release Notes

This release adds a whole **writing-assistance layer** to Nyx: autocomplete,
grammar correction, and AI rewriting that work in any text box on any site — plus
a **one-line installer/updater** so you can keep Nyx current straight from the
terminal, no backend required.

> ⚠️ After updating, please reload the extension in `brave://extensions`.
> This release requests new host permissions (LanguageTool, Datamuse, and the AI
> providers) so your browser may prompt you to allow them.

---

## What's New

### Typing assistant — `>typing`
- Autocomplete and spelling fixes as you type, in any text box, on any site.
  A small popup follows your caret: **Tab** accepts, **↑/↓** pick, **Esc** dismisses.
- **Local** mode downloads a ~50,000-word frequency dictionary for your language
  once (`>typing local fr`) and then works fully offline. **Online** mode uses the
  Datamuse API (English), no download.

### Grammar fixes — `>grammar`
- Free grammar and spelling checking as you type (LanguageTool, no key, many
  languages). Fixes appear in a popup you can apply one-by-one or all at once.
- **Auto-write** (`>grammar auto`) applies fixes silently as you type. Rich editors
  like Gmail keep their formatting — only the word under your caret is touched.

### AI rewrite, reply & summarize
- **Alt+R** fixes the current text box with your own AI; **Shift+Alt+R** polishes it
  for clarity. Right-click the ✦ button for tone presets (formal / casual / shorter).
- Works with **Groq, OpenAI, Gemini, Claude, DeepSeek, Grok, Kimi, Qwen**, or a local
  **Ollama** (no key). Set it up with `>ai <provider>` then `>ai key <your-key>`.
- **Alt+A** drafts a reply to a message you've selected; **Alt+W** summarizes the page.
- Your API key stays in the background service worker — pages never see it.

### Little extras
- **Personal dictionary** (`>dict`, or the `＋` in the grammar popup) — your own words
  are never flagged again and show up in autocomplete.
- **Typing stats** (`>typing stats`) — words typed, completions, fixes, and your
  most-corrected mistakes. Counting stays on-device.
- An optional AI-written line in the once-a-day briefing when a provider is set up.

### Install & update from the terminal
- **Windows:** `irm https://raw.githubusercontent.com/RDXFGXY1/nyx/main/install.ps1 | iex`
- **Linux / macOS:** `curl -fsSL https://raw.githubusercontent.com/RDXFGXY1/nyx/main/install.sh | sh`
- The same command installs and updates. Updates happen **in place**, so the
  extension keeps its identity and all your saved data; `js/config.js` is never
  overwritten, and the previous build is backed up automatically.

---

## Notes

Grammar checking sends box text to languagetool.org, and the AI features send text
to whichever provider you configure — that's inherent to how they work. The local
dictionary and typing stats never leave your machine.
