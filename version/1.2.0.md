# Nyx 1.2.0 — 2026-07-10

## Added
- **Random wallpaper** — every new tab can load a random wallpaper from a folder
  on disk, served by the backend:
  - settings → appearance: an on/off toggle plus a type picker
    (images only / videos only / both);
  - a hint row showing the folder the server reads and how many images/videos it
    found (recommended: keep them in **Pictures/wallpaper**);
  - a **"wallpapers folder"** input — point the server at **any** folder
    (`POST /api/wallpaper/folder`, persisted so it survives restarts);
  - backend `WallpaperController` (`/api/wallpaper/list`, `/api/wallpaper/file`)
    reading the chosen folder, path-traversal-safe, with seekable video serving;
  - the accent is extracted from the random wallpaper (CORS-safe), falling back to
    your saved wallpaper when the server is off or the folder is empty.
